# Q8 – Staff Class (Find Lowest Salary)

### Problem

Create class **Staff** with data members:

* `name`
* `bdate` (birth date)
* `designation`
* `salary`

Requirements:

1️⃣ Use **constructor** to initialize data
2️⃣ Create a method **display()**
3️⃣ Display the **staff member having the lowest salary**

---

# 💻 Java Program

```java
class Staff {

    String name;
    String bdate;
    String designation;
    double salary;

    Staff(String n, String b, String d, double s) {
        name = n;
        bdate = b;
        designation = d;
        salary = s;
    }

    void display() {
        System.out.println("Name: " + name);
        System.out.println("Birth Date: " + bdate);
        System.out.println("Designation: " + designation);
        System.out.println("Salary: " + salary);
    }
}

public class StaffTest {

    public static void main(String args[]) {

        Staff s[] = new Staff[3];

        s[0] = new Staff("Rahul", "10-02-2000", "Manager", 50000);
        s[1] = new Staff("Amit", "15-03-1999", "Clerk", 25000);
        s[2] = new Staff("Neha", "20-04-2001", "HR", 30000);

        int index = 0;

        for(int i = 1; i < s.length; i++) {

            if(s[i].salary < s[index].salary) {
                index = i;
            }
        }

        System.out.println("Staff with Lowest Salary:");
        s[index].display();
    }
}
```

---

# Program Flow

### Step 1 – Constructor Runs

Example:

```java
new Staff("Rahul","10-02-2000","Manager",50000)
```

Constructor automatically assigns:

```
name = Rahul
bdate = 10-02-2000
designation = Manager
salary = 50000
```

---

### Step 2 – Object Array

```
s[0] Rahul
s[1] Amit
s[2] Neha
```

---

### Step 3 – Find Lowest Salary

Compare salaries:

```
50000
25000 ✔ lowest
30000
```

So lowest = **Amit**

---

# Output

```
Staff with Lowest Salary:
Name: Amit
Birth Date: 15-03-1999
Designation: Clerk
Salary: 25000
```

# Important Concept – Constructor

A **constructor** is a special method that runs automatically when an object is created.

Example:

```java
Staff(String n, String b, String d, double s)
```

Called when:

```java
new Staff(...)
```

---

# Why Constructor is Useful

Without constructor you must write:

```
s[0].name = Rahul
s[0].salary = 50000
```

Constructor allows:

```
new Staff("Rahul",...,50000)
```

Cleaner code.

---

# Common Beginner Mistake

❌ Writing return type in constructor

Wrong

```java
void Staff(...)
```

Correct

```java
Staff(...)
```
