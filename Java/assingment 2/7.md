# Q7 – Student Result (Find Topper)

### Problem

Create two classes:

### 1️⃣ `Student`

Data members:

* `PRN_No`
* `name`
* `college_code`

---

### 2️⃣ `Stud_Result`

Data members:

* `PRN_No`
* `Sub_Code`
* `marks_Obtained`

Program should **find the topper whose total marks are maximum**.

---

# 💻 Java Program

```java
class Student {

    int prn;
    String name;
    int collegeCode;
}

class Stud_Result {

    int prn;
    int subCode;
    int marks;
}

public class Topper {

    public static void main(String args[]) {

        Student s[] = new Student[3];
        Stud_Result r[] = new Stud_Result[3];

        for(int i=0;i<3;i++) {
            s[i] = new Student();
            r[i] = new Stud_Result();
        }

        s[0].prn = 1;
        s[0].name = "Rahul";
        s[0].collegeCode = 101;

        s[1].prn = 2;
        s[1].name = "Amit";
        s[1].collegeCode = 101;

        s[2].prn = 3;
        s[2].name = "Neha";
        s[2].collegeCode = 101;

        r[0].prn = 1;
        r[0].subCode = 201;
        r[0].marks = 85;

        r[1].prn = 2;
        r[1].subCode = 201;
        r[1].marks = 90;

        r[2].prn = 3;
        r[2].subCode = 201;
        r[2].marks = 80;

        int max = r[0].marks;
        int index = 0;

        for(int i=1;i<3;i++) {

            if(r[i].marks > max) {
                max = r[i].marks;
                index = i;
            }
        }

        System.out.println("Topper Details:");
        System.out.println("PRN: " + s[index].prn);
        System.out.println("Name: " + s[index].name);
        System.out.println("Marks: " + max);
    }
}
```

---

# Program Flow

### Step 1 – Create Student Objects

```text
s[0] Rahul
s[1] Amit
s[2] Neha
```

---

### Step 2 – Store Marks

```text
Rahul → 85
Amit → 90
Neha → 80
```

---

### Step 3 – Find Maximum Marks

Logic:

```
max = first marks
```

Compare:

```
90 > 85 ✔
max = 90
```

Next

```
80 > 90 ❌
```

So topper is **Amit**.
