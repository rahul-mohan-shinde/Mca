# Q4 – Class `Square`

**Problem**

Create a class **Square** with:

**Data Member**

* `side`

**Methods**

* `area()`
* `perimeter()`

Create an **object of Square class** and test it.

---

# 💻 Java Program

```java
import java.util.Scanner;

class Square {

    int side;

    void area() {
        int a = side * side;
        System.out.println("Area of Square = " + a);
    }

    void perimeter() {
        int p = 4 * side;
        System.out.println("Perimeter of Square = " + p);
    }
}

public class TestSquare {

    public static void main(String args[]) {

        Scanner sc = new Scanner(System.in);

        Square s = new Square();   // object creation

        System.out.print("Enter side of square: ");
        s.side = sc.nextInt();

        s.area();
        s.perimeter();
    }
}
```

---

# Program Flow

### 1️⃣ Class is created

```
class Square
```

It contains

```
side
area()
perimeter()
```

---

### 2️⃣ Object is created

```
Square s = new Square();
```

Think like this:

```
Square → Blueprint
s → Real object
```

---

### 3️⃣ Assign value

```
s.side = 5
```

Object memory:

```
s
 └── side = 5
```

---

### 4️⃣ Method call

```
s.area()
```

Calculation:

```
area = side * side
5 * 5 = 25
```

---

### 5️⃣ Perimeter

```
s.perimeter()
```

Calculation:

```
perimeter = 4 * side
4 * 5 = 20
```

# Common Beginner Mistakes

❌ Forgetting object creation

```
Square s;
```

Correct

```
Square s = new Square();
```

---

❌ Trying to call method without object

Wrong

```
area();
```

Correct

```
s.area();
```
