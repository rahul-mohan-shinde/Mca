# Q2 – Check Character Type (Command Line Argument)

**Problem:**
Write a Java program that accepts **one character from command line** and checks:

1️⃣ Alphabet
2️⃣ Digit
3️⃣ Space

If it is **alphabet**:

* Check **capital or small**
* Convert it to **reverse case**

❗ We will **not use built-in functions** like `Character.isDigit()` etc.

---

# 💻 Java Program

```java
public class CheckChar {

    public static void main(String args[]) {

        char ch = args[0].charAt(0);

        if(ch >= 'A' && ch <= 'Z') {
            System.out.println("Capital Alphabet");

            char lower = (char)(ch + 32);
            System.out.println("Reverse case: " + lower);
        }

        else if(ch >= 'a' && ch <= 'z') {
            System.out.println("Small Alphabet");

            char upper = (char)(ch - 32);
            System.out.println("Reverse case: " + upper);
        }

        else if(ch >= '0' && ch <= '9') {
            System.out.println("Digit");
        }

        else if(ch == ' ') {
            System.out.println("Space Character");
        }

        else {
            System.out.println("Special Character");
        }

    }
}
```

# 🔎 Important Concept (ASCII Logic)

Characters have **ASCII values**.

| Character | ASCII |
| --------- | ----- |
| A         | 65    |
| Z         | 90    |
| a         | 97    |
| z         | 122   |

So checking:

```java
if(ch >= 'A' && ch <= 'Z')
```

means **capital alphabet**.

---

# 🔄 Reverse Case Logic

Difference between **A and a = 32**

Example

```
A = 65
a = 97
97 - 65 = 32
```

So

```
Capital → small = +32
Small → capital = -32
```

---

# Program Flow

```
Take character from args

Check

A-Z → Capital alphabet
a-z → Small alphabet
0-9 → Digit
' ' → Space
Else → Special character
```

# ⚠️ Beginner Mistake

❌ Writing

```java
char ch = args[0];
```

Correct

```java
char ch = args[0].charAt(0);
```

Because **args stores String not char**.
