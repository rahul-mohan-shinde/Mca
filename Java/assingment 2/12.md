# Q12 – Employee Payroll System

### Requirements

Create class **Employee** with **private data members**:

* `emp_id`
* `emp_name`
* `basic_sal`
* `hra`
* `medical`
* `pf`
* `pt`
* `net_sal`
* `gross_sal`

---

### Salary Formula

```
HRA = 50% of basic salary
PF  = 12% of basic salary
PT  = 200
```

```
Gross Salary = basic + hra + medical
Net Salary   = gross - (pt + pf)
```

---

# 💻 Java Program

```java
import java.util.Scanner;

class Employee {

    private int emp_id;
    private String emp_name;
    private double basic_sal;
    private double hra;
    private double medical;
    private double pf;
    private double pt;
    private double net_sal;
    private double gross_sal;

    static int count = 0;

    Employee() {
        emp_id = ++count;
    }

    Employee(String name, double basic, double medical) {

        emp_id = ++count;
        emp_name = name;
        basic_sal = basic;
        this.medical = medical;
    }

    void calculate() {

        hra = 0.5 * basic_sal;
        pf = 0.12 * basic_sal;
        pt = 200;

        gross_sal = basic_sal + hra + medical;
        net_sal = gross_sal - (pt + pf);
    }

    void display() {

        System.out.println("Employee ID: " + emp_id);
        System.out.println("Name: " + emp_name);
        System.out.println("Gross Salary: " + gross_sal);
        System.out.println("Net Salary: " + net_sal);
    }

    static void totalEmployees() {
        System.out.println("Total Employees: " + count);
    }
}

public class Payroll {

    public static void main(String args[]) {

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter number of employees: ");
        int n = sc.nextInt();

        Employee e[] = new Employee[n];

        for(int i = 0; i < n; i++) {

            System.out.println("\nEnter details of employee " + (i+1));

            System.out.print("Name: ");
            String name = sc.next();

            System.out.print("Basic Salary: ");
            double basic = sc.nextDouble();

            System.out.print("Medical Allowance: ");
            double medical = sc.nextDouble();

            e[i] = new Employee(name, basic, medical);

            e[i].calculate();
        }

        for(int i = 0; i < n; i++) {

            System.out.println("\nEmployee Details");
            e[i].display();
        }

        Employee.totalEmployees();
    }
}
```

# Program Flow

### Step 1 – Static Variable

```
static int count
```

Used to **count employees automatically**.

Example

```
Employee 1 → ID = 1
Employee 2 → ID = 2
Employee 3 → ID = 3
```

---

### Step 2 – Constructor

When object is created

```
new Employee(...)
```

Constructor runs and assigns **auto employee id**.

---

### Step 3 – Salary Calculation

Example

```
Basic Salary = 20000
Medical = 1000
```

Calculate:

```
HRA = 50% of 20000 = 10000
PF  = 12% of 20000 = 2400
PT  = 200
```

---

### Step 4 – Gross Salary

```
Gross = basic + hra + medical
```

```
= 20000 + 10000 + 1000
= 31000
```

---

### Step 5 – Net Salary

```
Net = gross - (pt + pf)
```

```
= 31000 - (200 + 2400)
= 28400
```

```

### 2️⃣ Static Variable

```
