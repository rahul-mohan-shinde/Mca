# Q5 – Employee Class (Sort by Salary)

**Problem**

Create a class **Employee** with:

* `name`
* `salary`

Create **5 employee objects**, then **sort them based on highest salary** and print them.

---

# 💻 Java Program

```java
class Employee {

    String name;
    int salary;

    void display() {
        System.out.println(name + " - " + salary);
    }
}

public class EmployeeSort {

    public static void main(String args[]) {

        Employee emp[] = new Employee[5];

        emp[0] = new Employee();
        emp[1] = new Employee();
        emp[2] = new Employee();
        emp[3] = new Employee();
        emp[4] = new Employee();

        emp[0].name = "Rahul";
        emp[0].salary = 30000;

        emp[1].name = "Amit";
        emp[1].salary = 45000;

        emp[2].name = "Neha";
        emp[2].salary = 25000;

        emp[3].name = "Rohan";
        emp[3].salary = 50000;

        emp[4].name = "Priya";
        emp[4].salary = 35000;

        Employee temp;

        for(int i = 0; i < emp.length - 1; i++) {

            for(int j = i + 1; j < emp.length; j++) {

                if(emp[i].salary < emp[j].salary) {

                    temp = emp[i];
                    emp[i] = emp[j];
                    emp[j] = temp;

                }
            }
        }

        System.out.println("Employees sorted by highest salary:");

        for(int i = 0; i < emp.length; i++) {
            emp[i].display();
        }
    }
}
```

---

# Program Flow

### 1️⃣ Create Employee Class

```java
class Employee
```

Members

```
name
salary
display()
```

---

### 2️⃣ Create Array of Objects

```java
Employee emp[] = new Employee[5];
```

Memory looks like:

```
emp[0]
emp[1]
emp[2]
emp[3]
emp[4]
```

---

### 3️⃣ Create Objects

```
emp[0] = new Employee();
emp[1] = new Employee();
```

Now every position has an **Employee object**.

---

### 4️⃣ Assign Data

Example

```
emp[0].name = Rahul
emp[0].salary = 30000
```

---

### 5️⃣ Sorting Logic

We compare salaries.

Example

```
30000 vs 45000
```

Swap

```
45000 comes first
```

So final order:

```
50000
45000
35000
30000
25000
```

---

# Example Output

```
Employees sorted by highest salary:

Rohan - 50000
Amit - 45000
Priya - 35000
Rahul - 30000
Neha - 25000
```

# Important Concept (Object Array)

Normal array:

```
int arr[]
```

Object array:

```
Employee emp[]
```

Each element stores an **object reference**.

---

# Common Mistake

❌ Only writing

```java
Employee emp[] = new Employee[5];
```

This **does not create objects**.

You must also write

```java
emp[0] = new Employee();
```
