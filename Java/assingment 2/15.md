# Q15 – Demonstrate `this` Keyword in Java

(This question again asks to demonstrate `this`, but usually teachers expect **different uses of `this`**.)

### Common Uses of `this`

1️⃣ Refer current class variable
2️⃣ Call current class method
3️⃣ Pass current object

Here we will demonstrate **variable reference + method call**.

---

# 💻 Java Program

```java
class Demo {

    int a;

    Demo(int a) {
        this.a = a;     // referring current object variable
    }

    void show() {
        System.out.println("Value of a = " + a);
    }

    void display() {
        this.show();    // calling current class method
    }
}

public class ThisKeywordDemo {

    public static void main(String args[]) {

        Demo d = new Demo(10);

        d.display();
    }
}
```

---

# Program Flow

### Step 1 – Object Creation

```java
Demo d = new Demo(10);
```

Constructor runs.

```
a = 10
```

---

### Step 2 – `this.a = a`

Meaning:

```
this.a → object variable
a → constructor parameter
```

So object stores

```
d.a = 10
```

---

### Step 3 – Method Call

```java
d.display();
```

Inside display

```
this.show();
```

This calls method **show()** of the same object.

---

# Output

```
Value of a = 10
```

---

# Visual Idea

```
Object d

d
 └─ a = 10
```

`this` always refers to **that current object**.

---

# All Uses of `this` (Remember for Viva)

| Use                     | Example        |
| ----------------------- | -------------- |
| Refer instance variable | `this.a = a`   |
| Call method             | `this.show()`  |
| Call constructor        | `this()`       |
| Pass object             | `method(this)` |
