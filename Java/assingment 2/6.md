# Q6 – Employee Salary Calculation

### Problem

Design a class **Employee** with:

**Data Members**

* `No`
* `Name`
* `Basic Salary`
* `DA = 12% of basic`
* `HRA = 25% of basic`
* `Professional Tax = 5% of basic`

**Methods**

1. `accept()` → Accept employee details
2. `calculate()` → Calculate net salary
3. `display()` → Display employee details

---

# 💻 Java Program

```java
import java.util.Scanner;

class Employee {

    int no;
    String name;
    double basic;
    double da, hra, pt, netSalary;

    void accept() {

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter Employee No: ");
        no = sc.nextInt();

        System.out.print("Enter Name: ");
        name = sc.next();

        System.out.print("Enter Basic Salary: ");
        basic = sc.nextDouble();
    }

    void calculate() {

        da = 0.12 * basic;
        hra = 0.25 * basic;
        pt = 0.05 * basic;

        netSalary = basic + da + hra - pt;
    }

    void display() {

        System.out.println("Employee No: " + no);
        System.out.println("Name: " + name);
        System.out.println("Basic Salary: " + basic);
        System.out.println("DA: " + da);
        System.out.println("HRA: " + hra);
        System.out.println("Professional Tax: " + pt);
        System.out.println("Net Salary: " + netSalary);
    }
}

public class EmployeeTest {

    public static void main(String args[]) {

        Employee e = new Employee();

        e.accept();
        e.calculate();
        e.display();
    }
}
```

### 3️⃣ Salary Calculation

```
DA = 12% of basic
   = 0.12 × 20000
   = 2400
```

```
HRA = 25% of basic
    = 0.25 × 20000
    = 5000
```

```
PT = 5% of basic
   = 0.05 × 20000
   = 1000
```

---

### 4️⃣ Net Salary

```
Net Salary = Basic + DA + HRA - PT
```

```
= 20000 + 2400 + 5000 - 1000
= 26400
```

# Common Mistake

❌ Calculating inside `display()`

Better practice:

```
accept()
calculate()
display()
```

Each method has **one responsibility**.
