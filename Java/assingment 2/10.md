# Q10 – Method Overloading (add two numbers & two strings)

### Problem

Create a method **add()** that can:

1️⃣ Add **two numbers**
2️⃣ Add **two strings**

Both methods have the **same name but different parameters**.

This is called **Method Overloading**.

---

# 💻 Java Program

```java
class AddDemo {

    int add(int a, int b) {
        return a + b;
    }

    String add(String a, String b) {
        return a + b;
    }

    public static void main(String args[]) {

        AddDemo obj = new AddDemo();

        int result1 = obj.add(10, 20);
        String result2 = obj.add("Hello ", "Java");

        System.out.println("Addition of numbers: " + result1);
        System.out.println("Addition of strings: " + result2);
    }
}
```

---

# Program Flow

### Step 1 – Object Creation

```
AddDemo obj = new AddDemo();
```

---

### Step 2 – First Method Call

```
obj.add(10,20)
```

Java chooses:

```
int add(int a, int b)
```

Result

```
30
```

---

### Step 3 – Second Method Call

```
obj.add("Hello ","Java")
```

Java chooses:

```
String add(String a, String b)
```

Result

```
Hello Java
```

---

# Output

```
Addition of numbers: 30
Addition of strings: Hello Java
```

---

# Important Concept – Method Overloading

Same method name
Different parameters

Example

```
add(int,int)
add(String,String)
```

Java decides **which method to call based on arguments**.

---

# Real World Example

Think of **print()** in Java.

```
print(int)
print(String)
print(double)
```

Same name but different inputs.

---

# Rules of Method Overloading

Method must differ in:

✔ number of parameters
✔ type of parameters

Not allowed:

❌ Only changing return type.

Wrong example

```
int add(int a,int b)
double add(int a,int b)
```
