# Q13 – Static Nested Class in Java

### Concept

A **Static Nested Class** is a **class inside another class**, and it is declared using the **`static` keyword**.

Important points:

* It **does not need an object of outer class**
* It can be accessed using **OuterClass.InnerClass**

---

# 💻 Java Program

```java
class Outer {

    static class Inner {

        void display() {
            System.out.println("Inside Static Nested Class");
        }

    }

}

public class StaticNestedDemo {

    public static void main(String args[]) {

        Outer.Inner obj = new Outer.Inner();

        obj.display();
    }

}
```

---

# Program Flow

### Step 1 – Outer Class

```
Outer
```

Inside it we created another class

```
Inner
```

But it is **static**.

---

### Step 2 – Object Creation

Because the class is static, we can directly write:

```java
Outer.Inner obj = new Outer.Inner();
```

We **do not need an Outer object**.

---

### Step 3 – Call Method

```
obj.display()
```

Output printed.

---

Inside Static Nested Class

# Important Difference

### Static Nested Class

```
static class Inner
```

Does **not require outer class object**

---

### Normal Inner Class

```
class Inner
```

Requires **outer class object**.

# Common Beginner Mistake

❌ Trying to create object like this

```
Inner obj = new Inner();
```

Correct

```
Outer.Inner obj = new Outer.Inner();
```
