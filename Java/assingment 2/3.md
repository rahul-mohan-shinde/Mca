# Q3 – Sort City Names in Ascending Order (Command Line Arguments)

**Problem:**
Accept **city names from command line arguments** and sort them in **ascending order (A → Z)**.

Example run:

```bash
java SortCities Pune Mumbai Delhi Nagpur
```

Expected Output:

```
Delhi
Mumbai
Nagpur
Pune
```

---

# 💻 Java Program (Without Built-in Sorting)

We will use **Bubble Sort**.

```java
public class SortCities {

    public static void main(String args[]) {

        String temp;

        for(int i = 0; i < args.length - 1; i++) {

            for(int j = i + 1; j < args.length; j++) {

                if(args[i].compareTo(args[j]) > 0) {

                    temp = args[i];
                    args[i] = args[j];
                    args[j] = temp;

                }
            }
        }

        System.out.println("Cities in Ascending Order:");

        for(int i = 0; i < args.length; i++) {
            System.out.println(args[i]);
        }
    }
}
```

# 🔎 Program Flow

Initial array

```
[Pune, Mumbai, Delhi, Nagpur]
```

First comparison

```
Pune vs Mumbai
Swap
```

Array becomes

```
[Mumbai, Pune, Delhi, Nagpur]
```

Then

```
Pune vs Delhi
Swap
```

Eventually sorted

```
[Delhi, Mumbai, Nagpur, Pune]
```

---

# Why We Used Two Loops

### Outer Loop

```
for(i)
```

Controls **number of passes**

### Inner Loop

```
for(j)
```

Compares **cities and swaps**

# ⚠️ Beginner Mistake

❌ Forgetting **temporary variable**

Wrong

```java
args[i] = args[j];
args[j] = args[i];
```

Correct

```java
temp = args[i];
args[i] = args[j];
args[j] = temp;
```
