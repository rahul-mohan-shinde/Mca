# Q14 – Inner Class in Java

### Concept

An **Inner Class** is a class **defined inside another class**, but **without the `static` keyword**.

Important points:

* It **requires an object of the outer class**
* It can access **all members of the outer class**

---

# 💻 Java Program

```java
class Outer {

    int x = 10;

    class Inner {

        void display() {
            System.out.println("Value of x = " + x);
        }

    }

}

public class InnerClassDemo {

    public static void main(String args[]) {

        Outer obj = new Outer();

        Outer.Inner in = obj.new Inner();

        in.display();
    }

}
```

---

# Program Flow

### Step 1 – Create Outer Object

```java
Outer obj = new Outer();
```

Object memory:

```
obj
 └─ x = 10
```

---

### Step 2 – Create Inner Class Object

```java
Outer.Inner in = obj.new Inner();
```

Important rule:

Inner class **needs outer class object**.

---

### Step 3 – Call Method

```
in.display()
```

Inner class prints the value of **outer variable `x`**.

---

# Output

```
Value of x = 10
```

---

# Visual Structure

```
Outer Class
   │
   ├─ variable x
   │
   └─ Inner Class
         └─ display()
```

Inner class can access **outer class variables directly**.

# Difference Between Q13 and Q14

| Feature             | Static Nested Class                   | Inner Class                              |
| ------------------- | ------------------------------------- | ---------------------------------------- |
| Keyword             | `static`                              | No static                                |
| Outer object needed | ❌ No                                  | ✔ Yes                                    |
| Object creation     | `Outer.Inner obj = new Outer.Inner()` | `Outer.Inner obj = outerObj.new Inner()` |

---

# Common Beginner Mistake

Wrong

```java
Outer.Inner in = new Outer.Inner();
```

Correct

```java
Outer obj = new Outer();
Outer.Inner in = obj.new Inner();
```
